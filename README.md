Scenario1:
         Association Rule Mining using the Apriori algorithm is used to discover frequent itemsets and meaningful relationships in transactional datasets such as grocery purchases. After preprocessing and converting data into one-hot encoded format, the algorithm identifies item combinations based on a minimum support threshold. It then generates association rules, which are evaluated using support, confidence, and lift. These metrics help determine the strength and usefulness of relationships between items. By analyzing different thresholds, strong and relevant rules can be identified, providing insights into customer buying patterns. Visualizations like bar charts and network graphs further help in understanding item associations.

Scenario2:
     Principal Component Analysis (PCA) is a dimensionality reduction technique used to transform high-dimensional numerical data into a smaller set of components while retaining maximum variance. After preprocessing and standardizing features, PCA extracts principal components and ranks them based on explained variance. This helps in selecting the optimal number of components, improving visualization, reducing complexity, and preserving important data patterns.
